Description

Project setup

  1. Install dependencies
Example :
$ pnpm ci
  1. Start your first migration (needs db to be started)
Example :
$ pnpm run db:migration:start

if you have problems try to use:

Example :
pnpm run db:gen
  1. Run your prisma server and app
Example :
$ pnpm run db:start && pnpm run start

Scripts

Start project

Example :
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# debug mode
$ pnpm run start:debug

# production mode
$ pnpm run start:prod

Linting and formating

Example :
$ pnpm run format

$ pnpm run lint

Prisma

Example :
# start in development mode
$ pnpm run db:start

# start migrations in dev (needed for work)
$ pnpm run db:migration:start

# reset migrations
$ pnpm run db:migration:reset

# generate in dev
$ pnpm run db:gen

# generate in production
$ pnpm run db:gen:prod

Tests

Example :
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
