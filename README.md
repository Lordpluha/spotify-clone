## Tech Stack
### Front
 - Next v15 App Router + Server Actions
 - TS
 - TailwindCSS + Module.css, clsx
 - Zod + React-hook-form
 - i18n
 - @tanstack/react-query + Socket.io
 - React-native
### Back
 - Nest
 - TS
 - MongoDB/PostgreSQL
 - RestAPI, SSE, Socket.io, Long-pooling, RabbitMQ
 - JWT, OAuth, CORS, CSP
 - Redis
### Infra
 - Prettier, ESLint, Lint-staged, husky.js
 - Pnpm, TurboRepo
 - github CI/CD, Sentry
 - Docker
